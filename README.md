# Pong-2D-Game
Windows Game <br>
Built Using Unity 2020.3.2f1 <br>
Dilo Game Academy's Assignment

## Player Control
<kbd>W</kbd> Player 1 move up <br>
<kbd>S</kbd> Player 1 move down <br>
<kbd>Arrow Up</kbd> Player 2 move up <br>
<kbd>Arrow Down</kbd> Player 2 move down <br>
